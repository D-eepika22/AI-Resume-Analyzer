\## What Makes This Project Different?



Most resume analyzers only work for Computer Science (CSE) students and provide generic ATS scores based on a fixed set of technical skills.



This project goes beyond that by supporting multiple domains and generating role-specific ATS analysis.



\##  Unique Features



\### 1. Multi-Domain ATS Scoring



Unlike traditional resume analyzers that focus only on software roles, this system supports:



\* Software Engineering

\* Web Development

\* AI / Machine Learning

\* Data Analytics

\* ECE Engineering

\* Electrical Engineering

\* Mechanical Engineering

\* Civil Engineering

\* MBA

\* BBA



Each role has its own skill database and ATS scoring criteria.



\### 2. Dynamic Skill Matching



The ATS score is not generated using a fixed keyword list.



The system automatically changes the required skills based on the selected job role and evaluates the resume accordingly.



Example:



\* An ECE student is evaluated on Embedded Systems, VLSI, IoT, MATLAB, Verilog, etc.

\* A Web Developer is evaluated on HTML, CSS, JavaScript, React, Node.js, etc.



\### 3. AI-Powered Resume Review



The project uses Large Language Models (LLMs) to provide:



\* Professional Summary

\* Strengths

\* Weaknesses

\* Missing Skills

\* Improvement Suggestions

\* Interview Questions

\* Recommended Projects



\### 4. ATS Score Breakdown



Instead of showing only a final ATS score, the system displays section-wise scoring for:



\* Skills

\* Experience

\* Projects

\* Education

\* Achievements

\* Formatting



This helps users understand exactly where they need improvement.



\### 5. Resume Improvement Generator



The application can generate an ATS-optimized version of the uploaded resume with improved content and formatting suggestions.



\### 6. Dual Resume Input System



Users can either:



\* Upload a PDF Resume

\* Paste Resume Text Directly



This makes the tool more flexible and user-friendly.



\### 7. Hackathon-Focused Real-World Solution



The project solves a real-world problem faced by students from different branches who often receive inaccurate ATS evaluations from tools designed only for software engineering roles.



This system provides more personalized and domain-specific resume analysis.
